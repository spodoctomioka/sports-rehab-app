# CLAUDE.md

# プロジェクト目的

スポーツ外傷・スポーツ障害のリハビリ支援アプリ

対象：

-   学生アスリート
-   一般スポーツ愛好家
-   保護者
-   医療知識が少ないユーザー
-   学生トレーナー
-   若手トレーナー

目的：

-   リハビリを理解しやすくする
-   行動につながる提案をする
-   離脱を減らす
-   医療者なしでも理解可能な表現にする
-   学生や若手トレーナーの学習にも利用できる内容にする

------------------------------------------------------------------------

# UI/UX方針

優先順位：

1.  科学的妥当性
2.  わかりやすさ
3.  入力負荷の低さ
4.  モバイルでの見やすさ
5.  見た目

ルール：

-   医学的内容は実在する論文・ガイドライン・教科書に基づく
-   根拠が曖昧な内容は採用しない
-   推測で補完しない
-   出典を必ず明記する
-   医学用語は最小限
-   必要時は短い説明を付ける
-   一画面の情報量を増やしすぎない
-   ボタン数を最小限にする
-   初見でも使えるUIを優先する
-   派手なアニメーション不要

------------------------------------------------------------------------

# コンテンツ方針

エビデンス重視

ルール：

-   実在する文献のみ使用
-   存在確認できない論文は引用禁止
-   論文名・著者・発表年を表示
-   可能ならDOIを表示
-   教科書引用時は書籍名と版数を表示
-   ガイドラインは発行団体を表示
-   論文内容をそのまま羅列しない
-   臨床や実際の行動に変換する

悪い例：

「遠心性収縮は筋腱複合体に作用し〜」

良い例：

「ゆっくりブレーキをかけながら行う運動は再発予防に役立つ可能性があります」

------------------------------------------------------------------------

# コード方針

-   必要なファイルのみ読む
-   全ファイル解析禁止
-   大規模リファクタリング禁止
-   既存構造を極力維持
-   変更範囲を最小限にする
-   不要な依存関係追加禁止
-   型エラー確認
-   ビルドエラー確認
-   他機能を壊さない

------------------------------------------------------------------------

# 修正時の出力形式

毎回以下のみ表示

【修正内容】 【理由】 【使用した出典】 【影響範囲】 【懸念点】
【次に改善候補】
